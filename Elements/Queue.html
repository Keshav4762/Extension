Queue - Essential Data Structure for Managing Data in a FIFO Manner

queue<int> q - Define a queue of integers.

q.push(a) - Enqueue an element 'a' into the queue.

q.front() - Access the element at the front

q.pop() - Dequeue the element at the front

Here, 'a' can be any data type. You can also initialize queues for characters and strings:

  • queue<char> q; - For characters.
  • queue<string> q; - For strings.
