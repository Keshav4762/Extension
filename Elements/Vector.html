Vector - One of the Most Important Data Structures

vector<int> v - Initialize a vector of integers.

v.push_back(a) - Insert an element 'a' into the vector.

Here, 'a' can be a number. You can also initialize vectors for characters and strings:

  • vector<char> v; - For characters.
  • vector<string> v; - For strings.

v.pop_back() - Remove the last element of the vector.

To sort a vector, use sort(v.begin(), v.end()).
