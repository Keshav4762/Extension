Set - Crucial Data Structure for Storing Unique Elements in Sorted Order

set<int> s - Define a set of integers.

s.insert(a) - Insert an element 'a' into the set.

s.erase(a) - Remove the element 'a' from the set.

s.find(a) - Find the iterator to the element 'a' in the set.

Here, 'a' can be any data type. You can also initialize sets for characters and strings:

  • set<char> s; - For characters.
  • set<string> s; - For strings.

s.empty() To check if the set is empty

s.size() To get the size of the set, use.
