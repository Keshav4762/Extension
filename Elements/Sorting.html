Sorting in C++

To sort an array, use sort(arr, arr + size).

To sort a string, use sort(str.begin(), str.end()).

To sort a linked list, you may need to convert it to a vector or deque, then sort it.

To sort a vector, use sort(vec.begin(), vec.end()).

To sort a deque, use sort(dq.begin(), dq.end()).

To sort a set, use set which automatically sorts its elements.

To sort a queue or a map, you need to transfer the elements to another container like a vector or a deque, then sort it.
