Map is used to store key-value pairs where each key is unique.

std::map<Key_Type, Value_Type> map_name; - Initializing a map.

Inserting elements into a map:

map_name[key] = value;

Accessing elements in a map:

map_name[key];

Iterating over elements in a map:

for (const auto& pair : map_name) {
std::cout << pair.first << " : " << pair.second << std::endl;
}
