Stack - Fundamental Data Structure for Last-In-First-Out (LIFO) Operations

stack<int> st - Define a stack of integers.

st.push(a) - Push an element 'a' onto the stack.

st.top() - Access the top element of the stack.

st.pop() - Pop the top element from the stack.

Here, 'a' can be any data type. You can also initialize stacks for characters and strings:

  • stack<char> st; - For characters.
  • stack<string> st; - For strings.
