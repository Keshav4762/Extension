Deque - Versatile Data Structure Supporting Insertion and Deletion at Both Ends

deque<int> dq - Define a deque of integers.

dq.push_back(a) - Insert an element 'a' into the back

dq.push_front(a) - Insert an element 'a' into the front

dq.front() - Access the element at the front

dq.back() - Access the element at the back

dq.pop_front() - Remove the element at the front

dq.pop_back() - Remove the element at the back
