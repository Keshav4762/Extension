String is used to store sequences of characters.

string str; - Declaring an empty string.

string str = "Hello"; - Initializing a string

string str = "Hello, World!";-Initializing a string

String Operations

str.size(); - Get the size of the string.

str.length(); - Get the length of the string (same as size).

str.empty(); - Check if the string is empty.

str.clear(); - Clear the contents of the string.

str.substr(start, length); - Extract a substring from the string.
